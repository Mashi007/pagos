- `INSTALAR_*` - Guías de instalación
- `COMANDOS_INSTALACION*` - Comandos necesarios para instalación
- `PASOS_INSTALACION*` - Pasos detallados de instalación
- `DEPLOYMENT_*` - Documentos de despliegue
